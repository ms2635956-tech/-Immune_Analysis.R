# 🧬 Genetic Analysis Project: Immune-Response-R

### **Objective:**
A professional R-pipeline to identify significant gene changes (Biomarkers) in immune cells after infection. 

### **Technical Steps:**
* **Data Prep:** Normalizing raw genomic data (Log2 transformation).
* **Filtering:** Using P-values and Fold-Change to pick out the most important genes (like IL-6 and TNF).
* **Plotting:** Visualizing results using high-quality **Volcano Plots**.

### **Why it matters:**
This project demonstrates the transition from **Clinical Veterinary Medicine** to **Medical Biotechnology** using Computational Biology.

**Tools:** R, ggplot2, ggrepel.
